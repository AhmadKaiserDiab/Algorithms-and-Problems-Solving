# Contents

### This Level Fouces on Building a strong Logic and Foundations and Solving Problems about numeric variables with 2 small Projects in the End of the Level built on Console

* the First Project simulates "Stone Paper Scissor" Game where the player can choose the number of rounds and display the final result and changing the screen colors base on Whether The player win or Lose

* the Second Project simulates a simple Math Game where the player can choose the number of Questions and it's Level and display the final result and changing the screen colors base on Whether The player Solved The Question or no
